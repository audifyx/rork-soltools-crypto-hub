# OGSCAN BOOK

# INTRODUCTION

OGSCAN is a Solana launch-intelligence system focused on reconstructing launch history and identifying canonical token origins.

Traditional scanners typically index visible chain state.

OGSCAN attempts to interpret:
- launch continuity
- migrations
- deployer behavior
- liquidity sequencing
- wallet relationships

through multi-signal ecosystem analysis.

---

# CORE INFRASTRUCTURE

The system focuses on:
- launch-state reconstruction
- token lineage mapping
- migration continuity
- deployer analysis
- liquidity interpretation
- ecosystem clustering

---

# WHY IT EXISTS

Solana launches are fragmented.

Problems include:
- duplicated tickers
- relaunches
- fake OG tokens
- migration confusion
- spam deployments

OGSCAN attempts to reconstruct historical launch context.
