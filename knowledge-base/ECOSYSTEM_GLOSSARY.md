# SOLTOOLS + OGSCAN ECOSYSTEM GLOSSARY

# CANONICAL ORIGIN
The highest-probability original launch token determined through multi-signal launch analysis.

---

# LAUNCH-STATE ANALYSIS
Interpretation of launch behavior using:
- liquidity flow
- migration continuity
- deployer behavior
- wallet relationships
- token sequencing

---

# TOKEN LINEAGE
The historical relationship between migrated or connected token deployments.

---

# MIGRATION CONTINUITY
Validation system used to determine whether a token migration represents continuation of an earlier ecosystem.

---

# WALLET GRAPH ANALYSIS
Behavioral clustering system mapping relationships between wallets and deployers.

---

# BUNDLE DETECTION
Detection of concentrated launch supply behavior.

---

# CRYPTO-NATIVE SOCIAL
Social infrastructure designed specifically for Web3 workflows.

# REALTIME COORDINATION
Instant communication and ecosystem participation systems optimized for crypto-native activity.
