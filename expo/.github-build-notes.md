# SolTools build notes

Use the Expo app from the `expo` directory.

## Local/dev

```bash
cd expo
bun install
bun run lint
bun run start-web
```

## Web export/build check

```bash
cd expo
bunx expo export --platform web
```

## Native build

Use EAS from the `expo` directory once Expo credentials are configured:

```bash
cd expo
bunx eas build --platform android
bunx eas build --platform ios
```

## Required env

The app expects Supabase values through the existing Supabase client configuration in `expo/lib/supabase.ts`.
